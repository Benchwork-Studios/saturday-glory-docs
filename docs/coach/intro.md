# Coach Guide

As a coach in Saturday Glory, you manage a college football program through a web browser. No app download required — the commissioner hosts the server and you connect from any browser on any device.

## What you do as a coach

Each week in the season follows the same pattern:

1. **Submit your game plan** — set your offensive and defensive tendencies for the upcoming game
2. **Take recruiting actions** — spend Action Points to recruit incoming players
3. **Mark yourself Ready** — signal to the commissioner you're done for the week
4. **Wait for the advance** — the commissioner runs the simulation when all coaches are ready

During the offseason, the rhythm shifts to recruiting and roster management instead of weekly game plans.

## What the portal shows you

| Page | What's there |
|------|-------------|
| **Dashboard** | Season, week, record, next opponent, last result, advance deadline |
| **Game Plan** | Offensive and defensive tendency sliders for the current week |
| **Recruiting** | Full prospect board with Action Point spending |
| **Roster** | Your team's full player list with ratings and class years |
| **Depth Chart** | Starter and backup assignments by position |
| **Schedule** | Your team's full season schedule with results |
| **Standings** | Conference standings and overall rankings |
