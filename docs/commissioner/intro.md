# Commissioner Guide

The commissioner is the person who runs the league. You own the Mac app, host the server, and control the simulation. Coaches connect via a web browser — they never touch your Mac.

## What the commissioner controls

| Responsibility | Where |
|---------------|-------|
| Starting the server and generating join codes | Mac app → League Hub |
| Assigning coaches to teams | Mac app → League Hub → Coach row |
| Locking and unlocking submissions | Mac app → League Hub → Season Control |
| Setting advance deadlines | Mac app → League Hub → Season Control |
| Advancing weeks and phases | Mac app → Season Hub |
| Removing coaches | Mac app → League Hub → Coach row |
| Resetting the league | Mac app → League Hub → Server Status |
| Sending league chat messages | Mac app → League Hub → League Chat |

## The commissioner workflow, week to week

Each week in Saturday Glory follows the same loop:

1. **Unlock submissions** — coaches can now submit game plans and recruiting actions
2. **Set a deadline** (optional) — gives coaches a target date
3. **Wait for all coaches to submit** — monitor the Submissions section in League Hub
4. **Lock submissions** — no further changes accepted
5. **Advance the week** — run the simulation in the Mac app
6. **Repeat**

The Submissions section shows you in real time who has submitted their game plan and how many recruiting actions they've logged. You can advance as soon as everyone is marked **Ready**, or after the deadline passes.
